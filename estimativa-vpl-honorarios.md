# Habilidade: Estimativa de VPL de Honorários

## 1. Objetivo
Calcular o VPL de honorários advocatícios (sucumbenciais ou contratuais) descontados a valor presente.

## 2. Instruções
1. Obter o valor base da condenação ou acordo.
2. Aplicar percentual de honorários (ex: 15%).
3. Estimar o tempo médio para recebimento (jurimetria por classe).
4. Aplicar fórmula VPL = valor_honorarios / (1+0,12)^t.
5. Se houver parcelamento, somar os fluxos descontados.

## 3. Exemplo
Condenação R$ 500.000, honorários 20% = R$ 100.000. Prazo 2 anos => VPL = 100.000 / (1,12^2) = R$ 79.719.

## 4. Agentes
A4.02, A4.06, A4.07
